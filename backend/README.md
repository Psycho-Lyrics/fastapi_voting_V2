### Ключевой функционал

| Определение функции           | Описание функции                                                                                                      |
| ----------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| Сессия                        | Инициализация, контроль и защита сессии пользователя                                                                  |
| Система голосований           | Создание, редактирование, поддержка, аналитика и прочие возможности для проведения и поддержки внутренних голосований |
| Панель администратора         | Панель с карт-бланшем на содержимое приложения                                                                        |
| Rate-limiter                  | Самописное приспособление для ограничения запросов целевых АПИ                                                        |
| Логирование                   | Процесс логирования жизненного цикла приложения                                                                       |
| Исключения                    | Описание концепции для отлова и обработки исключений                                                                  |
| Версионирование АПИ           | Описание версий приложения для мягкой и последовательной прогрессии функционала                                       |
| Проксирование трафика         | Описание реализации и требований от обратного прокси-сервера                                                          |
| Контейнеризация и оркестрация | Планирование шаблонов для образов, внедрение и поддержка мультисервисности                                            |
| CI/CD                         | Работа с GitHub Actions и планирование процесса автоматического развёртывания                                         |
| Планировщик                   | Работа планировщика. Детали связок с Redis и работой с процессами                                                     |
