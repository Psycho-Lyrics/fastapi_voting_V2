### Администратор получает возможность:
Сессия
	- Авторизация и регистрация администратора проходит в одноимённой панели
Голосования
	- Просматривать все существующие на платформе голосования, включая удалённые и отменённые
	- Применять условия фильтрации для перечня голосований: даты, статусы, архивное/активное, публичное/непубличное, конкретного отдела, удалено, отменено, создатель.
	- Производить поиск голосований по ключевым словам
	- Просматривать всю доступную платформе информацию про конкретное голосование
	- Создавать голосования
	- Редактировать активные и анонсированные голосования 
	- Удалять голосования 
	- Отменять голосования
Пользователи
	- Создавать новых пользователей
	- Редактировать информацию о пользователях
	- Удалять существующих пользователей
Центр уведомлений
	- Ознакомиться со списком всех запросов штату администраторов
	- Применить фильтрацию по условию запроса: регистрация глав отделов, смена почты
	- Редактировать вводные данные запросов. Например, информацию о главе отдела при регистрации.
### Глава отдела получает возможность:
Сессия
	- Авторизоваться с использованием личных учётных данных
		— Получать исчерпывающие сообщения о некорректности введённых учётных данных [[Технические вопросы#^4028eb|Защита от перебора адресов email]]
		— Сохранить сессию ("Запомнить меня")
	- Регистрироваться с использованием личных учётных данных [[Технические вопросы#^536e56|Процесс регистрации глав отдела]]
		— Избежать необходимости указывать отчество из культурных соображений
		— Получить гарантию защиты от человеческого фактора: поле для подтверждения пароля
	- Восстановить доступ к существующей учётной записи посредством подтверждения через почту и прямой интервенции лица-администратора [[Технические вопросы#^b5ea08|Участие администратора в опасных процедурах]] 
	- Прервать сессию
Голосования [[Технические вопросы#^8e9769|Возможности главы отдела]]
	- Просматривать доступные для участия голосования 
	- Производить поиск по голосованиям
	- Применять фильтры к голосованиям: дата регистрации/начала/окончания, статус: этап регистрации/сбора голосов/окончания/архивное. Собственные
	- Ознакомиться с общими деталями о **предстоящем/активном/прошедшем** голосовании
	- Получить пригласительную ссылку и QR-code
	- Создавать голосование с конкретными параметрами:
		— Дата начала регистрации -> дата окончания регистрации
		— Дата начала сбора голосов -> дата окончания сбора голосов
		— Дата подведения и публикации итогов -> дата архивации голосования
	- Принять участие в голосовании:
		— В любой момент подписаться/отписаться от рассылок уведомлений по конкретному голосованию 
		— Оставить заявку на регистрацию в голосовании с возможностью определить желание получить уведомление на email о начале этапа сбора голосов
		— Оставить свой голос 
	- Редактировать **предстоящие и активные голосования** [[Технические вопросы#^5e3538|Возможности в редактировании]]
	- Удалять голосования [[Технические вопросы#^a7194f|Мягкое удаление голосований]]
Личный кабинет 
	- Ознакомиться с данными своей учётной записи: имя, фамилия, отчество(если было указано) и адрес электронной почты 
	- Изменить данные учётной записи: имя, фамилия, отчество(если было указано) и адрес электронной почты
	- Отслеживать доступные для участия голосования
	- Отслеживать собственные голосования
	- Ознакомиться с оконченными голосованиями
### Сотрудник получает возможность:
Сессия
	- Авторизоваться с использованием личных учётных данных
		— Получать исчерпывающие сообщения о некорректности введённых учётных данных
		— Сохранить сессию ("Запомнить меня")
	- Регистрироваться c использованием личных учётных данных
		— Избежать необходимости указывать отчество из культурных соображений
		— Получить гарантию защиты от человеческого фактора: поле для подтверждения пароля
	- Восстановить доступ к существующей учётной записи посредством подтверждения через почту 
	- Прервать сессию
Голосования
	- Просматривать публичные голосования без необходимости авторизоваться
	- Производить поиск по голосованиям
	- Применять фильтры к голосованиям: дата регистрации/начала/окончания, статус: этап/регистрации/сбора голосов/окончания/архивное
	- Ознакомится с общими деталями о **предстоящем/активном/прошедшем** голосовании без необходимости принимать участие в голосовании
	- Получить пригласительную ссылку и QR-code без необходимости принимать участие в голосовании [[Технические вопросы#^35b31c|Реферальная система]]
	- Принять участие в голосовании:
		— В любой момент подписаться/отписаться от рассылок уведомлений по конкретному голосованию 
		— Оставить заявку на регистрацию в голосовании с возможностью определить желание получить уведомление на email о начале этапа сбора голосов  [[Технические вопросы#^8ea146|Мануальная/автоматическая модерация заявок]]
		— Оставить свой голос 
Личный кабинет
	- Ознакомиться с данными своей учётной записи: имя, фамилия, отчество(если было указано), отделы сотрудника и адрес электронной почты [[Технические вопросы#^8eb026|Цензурирование почты и пароль для отображения]]
	- Изменить данные учётной записи: имя, фамилия, отчество(если было указано), отделы[[Технические вопросы#^f5180f|Опасные операции]] и адрес электронной почты [[Технические вопросы#^27a01a|Роль и включение резервного email]] 
	- Отслеживать доступные для участия голосования
	- Ознакомиться с оконченными голосованиями
